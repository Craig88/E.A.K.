CSS describes the design of a HTML page -
fonts, colours, and shapes.

The CSS is this bit here, inside the <style> tag.

A <style> tag is used to put CSS in HTML document, so it can change the page's design.

The word body tells the browser that the bits between the curly brackets are for the whole page.

background is saying that we want to change the background.
